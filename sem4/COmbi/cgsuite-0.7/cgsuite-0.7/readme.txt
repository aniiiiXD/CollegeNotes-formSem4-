To start Combinatorial Game Suite:

- First, make sure that you are running Java version 1.5.0 or later (you can
  type  java -version  to check).  You can always download the latest version
  (currently 1.6.0) from:  http://java.sun.com/

- Extract the cgsuite-0.7.zip archive.

- Change to the cgsuite-0.7/ directory and:
  * If you're using Windows, double-click the  cgsuite.jar  file.
  * If you're using Unix or Linux, type:  java -jar cgsuite.jar
  
- Then follow the instructions on the screen, and you're ready to go!  For a
  brief introduction and tutorial, select "Getting Started" from the Help menu.
